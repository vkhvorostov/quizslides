# ВВЕДЕНИЕ
## QuizSlides
Веб-конструктор интерактивных презентаций с онлайн-конференциями

Дата начала разработки: 2026г.

Основные возможности:
1. Создание и редактирование презентаций
2.  Проведение интерактива с аудиторией (опрос, облако слов)

Поддерживаемые языки: русский, английский (в разработке)

## Быстрый старт
### Создание и активация виртуального окружения
    python -m venv venv
###### Для Windows:
    venv\Scripts\activate
###### Для macOS/Linux:
    source venv/bin/activate

### Установка зависимостей
    pip install -r requirements.txt

### Настройка переменных окружения
Скопируйте файл `.env.example` в `.env` и укажите в нём реальные параметры подключения к базе данных:

    cp .env.example .env

После этого откройте `.env` и замените значения `DB_NAME`, `DB_USER`, `DB_PASSWORD`, `DB_HOST` и `DB_PORT` на собственные.

### Запуск проекта
    python manage.py runserver
