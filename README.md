# QuizSlides – интерактивная образовательная платформа

Проект на Django для проведения лекций, семинаров и студенческих мероприятий с интерактивными возможностями.

## Что было сделано

### 1. Интеграция HTML/CSS с Django
- Добавлен готовый HTML-шаблон главной страницы (`html.html`) в проект Django.
- Создана папка `style/` с кастомными CSS-стилями (`style.css`).
- Настроены статические файлы в `settings.py` (`STATICFILES_DIRS`, `STATIC_ROOT`).

### 2. Главная страница
- Создано представление `HomeView` в `quizslides/views.py`.
- Шаблон `templates/home.html` адаптирован под Django (использует наследование от `base.html`).
- Добавлены интерактивные элементы:
  - Поле для ввода кода участника.
  - Кнопка «Присоединиться».
  - Секция «Возможности платформы» с четырьмя стилизованными квадратами.

### 3. Единый дизайн для аутентификации
- Обновлён базовый шаблон `templates/base.html`:
  - Добавлены Bootstrap 5 и кастомные стили.
  - Навигационная панель с логотипом, кнопками «Войти» и «Регистрация».
  - Адаптивный дизайн и эффекты при наведении.
- Шаблоны входа (`login.html`) и регистрации (`signup.html`) переработаны:
  - Карточная структура с тенями и отступами.
  - Согласованные с главной страницей цвета и шрифты.

### 4. Рабочие кнопки навигации
- Кнопки «Войти» и «Регистрация» на главной странице и в навигационной панели ведут на соответствующие страницы аутентификации.
- Для авторизованных пользователей отображается приветствие и кнопка «Выход».

### 5. Визуальные улучшения
- **Квадраты «Возможности платформы»**:
  - Увеличены до 280px в высоту.
  - Светло-серая цветовая гамма (градиент `#e5e7eb` → `#f3f4f6`).
  - Тени, скругления, эффект подъёма при наведении.
- **Логотип**:
  - Загружен по ссылке и сохранён как `static/quizslides_logo.png`.
  - Интегрирован в навигационную панель (высота 40px).

### 6. Настройка URL-маршрутов
- Корневой URL (`/`) теперь отображает главную страницу вместо редиректа на вход.
- Маршруты аутентификации доступны по `/accounts/login/` и `/accounts/signup/`.

## Структура проекта

```
Django_glav/
├── accounts/          # Приложение аутентификации
├── quizslides/        # Настройки проекта
├── static/            # Статические файлы (логотип)
├── style/             # Кастомные CSS-стили
├── templates/         # HTML-шаблоны
│   ├── accounts/      # Шаблоны входа и регистрации
│   ├── base.html      # Базовый шаблон
│   └── home.html      # Главная страница
├── web/               # Дополнительные HTML-страницы
├── manage.py
└── README.md
```

## Запуск проекта

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Примените миграции:
   ```bash
   python manage.py migrate
   ```

3. Запустите сервер разработки:
   ```bash
   python manage.py runserver
   ```

## Технологии

- **Backend**: Django 3.2.16
- **Frontend**: Bootstrap 5, кастомный CSS
- **База данных**: SQLite3
- **Статические файлы**: Django Staticfiles
